#  Blinkit Grocery Sales Analysis using Tableau

##  Project Overview
This project analyzes grocery sales data for Blinkit to understand sales trends, customer behavior, and outlet performance using Tableau.

##  Tools Used
- Tableau
- Data Analysis

##  Key Insights
- Identified top-selling products and categories  
- Analyzed sales across different outlet types and locations  
- Studied impact of outlet age on sales  
- Evaluated customer preferences using product attributes  

##  Dashboard
Interactive dashboard with filters and multiple visualizations.

##  Files
- Blinkit.twbx  
- Grocery_Sales.csv  

##  How to Use
Download the file and open in Tableau Desktop.
